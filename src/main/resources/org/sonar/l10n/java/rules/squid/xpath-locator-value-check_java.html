Locators based on css should not be tied to page layout. Page layout can be changed frequently. Xpath locators which does not start with // (relative) and having a "/" defines it to be tied to page layout.

Good locators are unique, descriptive and unlikely to change.

Noncompliant Code Example

  By.xpath("/confirmation/book/message[text()]"); // Noncompliant

Compliant Solution

  By.xpath("//*[@id="element_id"]");
